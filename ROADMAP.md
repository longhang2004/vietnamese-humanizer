# Roadmap

## 0.2.x / Post-0.2

- Conduct evaluations with native Vietnamese speakers across diverse regions and domains.
- Measure inter-annotator agreement on benchmarks and log disagreement instead of forcing a single answer.
- Add domain-specific glossaries and longer prose fixtures.
- Calibrate linter thresholds using licensed, verified corpus data.

## 0.3

- Support custom project style guide configuration directly via the CLI.
- Add multi-model and human-annotator baseline benchmarks.
- Report precision metrics by pattern, domain, and register.
- Research handling bilingual text and subtitles.

## 1.0

- Stabilize data schemas and versioning policy.
- Publish inter-annotator evaluated benchmarks.
- Provide comprehensive integration docs for diverse agent clients.

*The roadmap explicitly excludes features designed for evading AI detectors.*
