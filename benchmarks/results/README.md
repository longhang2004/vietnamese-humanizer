# Trạng thái baseline

Repository chưa có human-reviewed baseline. Thư mục này chỉ nhận kết quả của đợt review có reviewer, system, prompt và phạm vi được ghi rõ.

Không dùng điểm tự sinh hoặc điểm minh họa làm kết quả thật. Có thể sao chép cấu trúc trong `../expected/README.md`, chấm độc lập rồi validate bằng benchmark runner.
