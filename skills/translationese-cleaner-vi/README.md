# translationese-cleaner-vi

Skill xử lý câu tiếng Việt bám trật tự và cliché tiếng Anh. Nó không tự động thay thuật ngữ và không sửa các cấu trúc pháp lý chỉ vì chúng trang trọng.

Ví dụ yêu cầu: `Làm sạch văn phong dịch của bản này, đối chiếu bản tiếng Anh và giữ nguyên tên tính năng.`

Catalog: `patterns/translationese.yml`. Bộ ví dụ đa miền: `examples/examples.jsonl`.
