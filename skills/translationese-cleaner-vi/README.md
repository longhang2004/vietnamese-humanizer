# translationese-cleaner-vi

Skill xử lý câu tiếng Việt bám trật tự và cliché tiếng Anh. Skill không tự đổi thuật ngữ hoặc sửa cấu trúc pháp lý chỉ vì chúng trang trọng.

Ví dụ yêu cầu: `Làm sạch văn phong dịch của bản này, đối chiếu bản tiếng Anh và giữ nguyên tên tính năng.`

Catalog nằm trong `patterns/translationese.yml`. Bộ ví dụ đa miền nằm trong `examples/examples.jsonl`.
