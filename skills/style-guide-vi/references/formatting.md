# Định dạng

- Giữ cấp heading tuần tự. Không nhảy từ H2 xuống H4 nếu không có lý do.
- Dùng bullet cho mục song song; dùng numbered list khi thứ tự có ý nghĩa.
- Các bullet cùng danh sách nên có cấu trúc ngữ pháp tương đồng.
- Không bôi đậm mọi từ khóa. Chỉ nhấn chỗ giúp quét nội dung.
- Dùng inline code cho command ngắn, path, option và identifier; dùng code block cho đoạn nhiều dòng.
- Dấu câu cuối bullet theo một quy ước trong cùng danh sách.
- Giữ nội dung rất ngắn dưới heading hiện có, thay vì tạo nhiều heading một dòng.
