# Vietnamese Writing Skills - Frontend UI

Next.js (App Router + TypeScript + Tailwind CSS) user interface for testing Vietnamese writing style signals, requesting Gemini rewrites, and submitting style contribution cases.

## Quick Start (Local Development)

1. Install dependencies:
   ```bash
   npm install
   ```

2. Copy local environment settings:
   ```bash
   cp .env.local.example .env.local
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

   The web app will open at `http://localhost:3000`.

## Production Build

```bash
npm run build
```
